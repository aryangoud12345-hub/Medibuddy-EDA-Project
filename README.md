# Exploratory Data Analysis (EDA) ‚Äì Health Insurance Dataset

## üìå Project Overview

This project performs an **Exploratory Data Analysis (EDA)** on a health insurance dataset to understand how demographic and lifestyle factors influence **insurance charges**. The analysis focuses on identifying patterns, correlations, outliers, and cost-driving variables that can support **data-driven business and policy decisions**.

The project is suitable for:

* Data Analytics / Data Science portfolios
* Academic assignments
* Insurance risk analysis use cases

---

## üìä Dataset Description

The dataset contains information about individuals and their medical insurance charges. Key variables include:

| Column         | Description               |
| -------------- | ------------------------- |
| age            | Age of the individual     |
| sex            | Gender of the individual  |
| bmi            | Body Mass Index           |
| children       | Number of dependents      |
| smoker         | Smoking status (Yes/No)   |
| region         | Residential region        |
| charges_in_INR | Medical insurance charges |

---

## üîç EDA Objectives

* Understand the distribution of insurance charges
* Identify key factors influencing medical costs
* Compare charges across risk groups
* Detect outliers and high-cost customers
* Support feature selection for predictive modeling

### 3Ô∏è‚É£ Top 10 Highest Insurance Charges (Bar Chart)

## üí° Key Takeaways

* **Smoking is the most influential factor** affecting insurance charges
* **Age and BMI** are important secondary drivers
* Insurance costs are **highly skewed**, driven by a small high-risk group
* Preventive healthcare and wellness programs can reduce long-term costs
